* [ ] 进程

  ```
  进程是系统独立运行且作为资源分配的最小单位
  ```

* [ ] 程序运行模式

  > 单用户多任务：一个用户可以同时运行多个进程
  >
  > 多用户多任务：多个用户可同时登录，并同时进行多任务处理

* [ ] 系统优化

  > 时空复用: 一个进程调用多个资源，当某个资源空闲的时候，转而为其他进程所用。
  >
  > 空分复用: 若在30m的机器上运行100m的程序，是先跑30m，然后30m退出，从硬盘中在导入30m,反复如此。

* [ ] 程序的顺序执行

  > 需遵循3个原则，1.顺序性，2.封闭性，3.可重入性。

* [ ] 时分复用和资源共享是进程可并发执行的两个必要条件。

* [ ] 若a的运行需要b先完成，即 b-->a,则说明b->a存在前驱关系，b为a的直接前驱，a为b的直接后继，没有前驱的点称为零点，在前驱图中，不能存在循环
